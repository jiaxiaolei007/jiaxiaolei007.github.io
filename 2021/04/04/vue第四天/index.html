代码中的id为歌曲id或者歌单id；server是音乐平台，添加netease，tencent，kugou等中的一个即可；type可以填写song, playlist, album, search, artist等中的一个，这三个属性为必须值。其他的属性含义如下：

属性 默认值 描述
autoplay false 自动播放，移动端不支持
fixed false 固定模式
mini false 迷你模式
list-folded false 歌词格式类型
mutex true 开启时，如果同页面有其他 aplayer 播放，该播放器会暂停
volume 0.4 播放器音量
theme #FADFA3 播放器风格色彩
preload auto 音乐文件预载入模式，可选项： none, metadata, auto
